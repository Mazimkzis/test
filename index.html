Изучаем веб-разработку

HTML - структура сайта

HTML - это язык разметки, который определяет структуру веб-страницы.

<div class="container">
    <h1>Заголовок</h1>
    <p>Параграф текста</p>
</div>

CSS - стили сайта

CSS отвечает за внешний вид элементов на странице.

.container {
    max-width: 800px;
    margin: 0 auto;
    padding: 20px;
}

JavaScript - интерактивность

JavaScript добавляет интерактивность на веб-страницу.

function showMessage() {
    alert('Привет, мир!');
}
